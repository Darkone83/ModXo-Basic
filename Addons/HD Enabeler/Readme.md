# HD Enabeler Addon

This board is based off TeamUIX's Epimetheus with some slight modifications to interface with the ModXo-Basic board

For more information on Epimetheus visit: https://github.com/OfficialTeamUIX/Epimetheus

### Builders notes

The value for L1 is a 200ohm ferrite bead in a 0805 footprint

#### Notice

We do not provide any firmware fot the STM32. You will need to either code your own or obtain a firmware image legally
